# Real Estate Database System 🏢

## 📌 Overview
This project is a Real Estate Database System built using SQL Server.  
الفكرة الأساسية هي تصميم نظام لإدارة:
- Builders
- Owners
- Houses
- Divisions
- Rooms
- Room Features

المشروع بيركّز على Database Design + Relationships + Querying بشكل عملي.

---

## 🎯 Objective
The goal of this project is to:
- Design a normalized database schema
- Define relationships between entities
- Implement real-world queries using SQL
- Use stored procedures to solve business problems

---

## 🧠 Key Concepts
- One-to-Many Relationships  
  (Owner → Houses, Division → Houses)

- Many-to-Many Relationships  
  (Rooms ↔ Features)

- Constraints & Data Integrity

- Real-world business logic implementation
